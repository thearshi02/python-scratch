#string-
string is the sequence of character.

#string indexing and slicing-
Every character in a string has an index starting at 0. 

indexing- You can get a single character using square bracket ex-txt[0].
Slicing- you can extract a subsection of a string using the syntax text[start:stop]. Note that the stop index is exclusive

#Essential String methods-
Python has built-in methods to manipulate text. SInce string are immutable, these methods always return a new modified string.

1- .upper()
2- .lower()
3- .strip()
4- .replace("old","new")
